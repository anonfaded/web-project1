Markdown Example

This is a sample Markdown document that demonstrates various Markdown syntax elements.

Headings

You can create headings using # symbols:

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

Emphasis

You can make text italic by using asterisks or underscores: - _italic_ - *italic*

You can make text bold by using double asterisks or underscores: - __bold__ - **bold**

Lists

Unordered List

  • Item 1
  • Item 2
    • Subitem 2.1
    • Subitem 2.2
  • Item 3

Ordered List

  1. First item
  2. Second item
  3. Third item

Links

You can create links by using square brackets for the link text and parentheses for the URL.

Images

You can embed images using an exclamation mark, square brackets for alt text, and parentheses for the image URL.

Code

You can include inline code by wrapping it in backticks: console.log("Hello, Markdown!");

You can also create code blocks using triple backticks:

```javascript function greet(name) { console.log(“Hello,” + name + “!”); }
